# Podcast RSS Analysis
Analyze podcast RSS feed metadata

## Setup
Ensure you have `pip` installed. Run `pip install -r requirements.txt`.

## Usage:
```
$ python parse.py https://www.npr.org/rss/podcast.php?id=510289
Processing 300 entires
Day Monday: 1
Day Tuesday: 2
Day Friday: 75
Day Wednesday: 110
Day Thursday: 39
Day Saturday: 73
```
